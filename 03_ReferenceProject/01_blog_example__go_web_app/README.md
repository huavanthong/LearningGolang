## Build
To build this example:
```
>go build
```

## Start-server
```
>./birdedia.exe
```

## Run
To run on brower
```
http://localhost:8080/assets/
```

To run by curl command
```
curl http://localhost:8080/assets/ -d {fill your data}
```

## Test
To run tests
```
>go test ./...
```
